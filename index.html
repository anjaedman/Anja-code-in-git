Mina git-kommandon!

Specifika git-kommandon är:

  • git status: kollar om filer är sparade + commitade
  • git log visar gjorda commiter
  • git add . : vi sparar nya ändringar
  • git commit -m "nytt meddelande":
  • git push : vi skickar våran sparning/ commit till cloud-servern
  • git pull : hämtar fjärr-repot
